# Layout Reference

- Dark full-window workbench with fixed side rails.
- Left rail contains the brand, input metadata, file drop/browse target, effect list, collapsed presets, and small footer links.
- Center preview is large, black, mostly unframed, and labeled at the top center with the active effect and backend.
- Right rail is one scrollable panel containing effect settings, adjustments, color, processing, post-processing, and export.
- Export format selection uses compact tiles: PNG, JPEG, GIF, WebP, Video, SVG, Text, and Three.js.
- Preview zoom and pan are viewport-only controls and must not change export resolution.
